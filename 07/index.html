Input Devices


I decided to use the one of the infrared sensors for this assignment. My goal was to measure the number of rotations my fan would spin. After connecting an IR sensor and an IR-LED to the metro board, I opened the serial plotter and had it display what the sensor was reading. This fan had 3 blades, and every time a blade passed between the sensor and the LED, it would drop the graph. based on my setup, the graph would read around 100 if it was unblocked by a fan blade, or 30 if it was blocked. I adjusted the arduino code to count every third spike and it seemed to be successful.

The strip LED looked fun and interesting to use. I started with the basic preset program but adjusted the arduino to have the lights bounce back and forth. I then added several additional colors and set them to run sequentially.

toggle

As I am planning to use two stepper motors for my final project, I decided it would be a good idea to practice programming them to run together. I learned there are two ways to run the motors: either to have them move one step at a time and adjust the delay, or to set a speed and adjust the number of steps. I decided against using the first method because the motor seemed to stutter if my delay was too short. Luckily for my final project, the motors should only need too move in the same direction at the same rate, so I was able to connect both through the breadboard to the same arduino ports.

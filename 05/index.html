Programable Electronics


My goal for this assignment was to assign each primary color of the RGB LED to a different button. My first attempt to the left failed because the red LED provided less resistance than the blue and green ones.

I redesigned the entire layout of the breadboard and adjusted the code signifigantly. While the old version had one current running straight into the buttons and led, this new version separated them. Here is the basic idea for the code below: each button and each primary LED color is assigned a different pin on the microcontroller. An if/esle statement determines which of the 8 possible combinations of on and off the buttons are and sends individual current to the corresponding color.
